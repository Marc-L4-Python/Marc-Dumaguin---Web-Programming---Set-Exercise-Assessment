HTML Quick Reference Guide

Headers

The header element represents a container for introductory content or a set of navigational links. A header element typically contains:

  • One or more heading elements (Headings 1 through 6)
  • Logo or Icon
  • Authorship Information

Paragraphs

The paragraph tag defines a paragraph. Browsers automatically add a single blank line before and after each paragraph element.

Phrasing Elements

Phrase elements add structural information to text fragments. A few examples of phrase elements with their meanings are the following.

  1. abbr - indicates an abbreviated form like: pvt. inc. etc.
  2. code - designates a fragment of computer code.
  3. b - styles the text in bold fonts by usage and convention.
  4. i - styles the text in italic form by usage and convention.
  5. mark - highlights the text in order to be easily referenced. (HTML5 only)

Div Element

A div element configures a structural block area or "division" on a web page with empty space above and below. It can also contain other block display elements, including other div elements.

Navigation Tag

The nav tag defines a set of navigation links. Notice that NOT all links of a document should be inside a "nav" element. The "nav" element is intended only for major blocks of navigation links. Browsers, such as screen readers for disabled users, can use this element to determine whether to omit the initial rendering of this content.

Main Element

The main tag specifies the main content of a document. The content inside the "main" element should be unique to the document. It should not contain any content that is repeated across documents such as sidebars, navigation links, copyright information, site logos, and search forms.

Footer Element

The footer tag defines a footer for a document or section. A footer element typically contains:

  • Authorship Information
  • Copyright Information
  • Contact Information
  • Sitemap
  • Back to Top Links
  • Related Documents
